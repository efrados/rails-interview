# rails-interview / TodoApi

This is a simple Todo List API built in Ruby on Rails 7

## Build

To build the application:

`bin/setup`

## Run the API

To run the TodoApi in your local environment:

`bin/puma`

## Test

To run tests:

`bin/rspec`

### Useful commands
* To compile styles
yarn build:css
* To compile js
yarn build
